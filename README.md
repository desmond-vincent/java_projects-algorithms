# Java Projects

A collection of practical Java projects demonstrating various programming concepts and techniques.

## Repository Structure
```
java_projects/
├── algorithms/
├── webscraper/
├── dice_roller/
└── ...
```

## Projects

### Algorithms
Implementation of common data structures and algorithms for technical interview preparation and problem-solving practice.

### Web Scraper
Web scraping application built with Jsoup for extracting and parsing data from websites.

### Dice Roller
Simulates dice rolling with customizable options for various tabletop games and probability experiments.

## Getting Started

### Prerequisites
- Java JDK 11 or higher
- Maven (for dependency management)

### Running Projects
Each project contains its own main class. Navigate to the project directory and run:
```bash
javac ProjectName.java
java ProjectName
```

## Technologies
- Java
- Jsoup (Web Scraper)
- Maven
